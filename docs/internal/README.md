# Internal Documentation

Dieser Bereich ist fuer interne Projektarbeit, Audit-Nachweise und historische Materialien vorgesehen.
Er gehoert nicht zur primaeren Produktbuehne des Repositories.

## Inhalte
- `legacy/`
  - Historische Deliverables und Abgabe-Artefakte
- `reference/`
  - Umfangreiche technische Referenzdokumente aus frueheren Projektstaenden
- `reports/`
  - Diagnose- und Audit-Reports

## Hinweis
Die in `internal/` enthaltenen Dateien sind fuer Nachvollziehbarkeit und Migration vorhanden, aber nicht als
zentrale Einstiegsdokumentation fuer externe Nutzer gedacht.
